Humble iCE

FPGA Development Board

Humble iCE is a cost optimized FPGA development board based on the Lattice iCE40UP5k chip, designed to make FPGA education more accessible. It pairs the FPGA with an RP2040 for programming and communication.

Mico

USB Microphone

Mico is an open source PDM to USB microphone based on the Raspberry Pi RP2040 chip.

More Information

SnapVCC

Portable Power Supply

snapVCC is a highly portable and convenient power supply for your electronics projects. It's designed to snap right on to a 9 V battery and give you 3.3 V or 5 V power wherever you need it.

More Information

Bluey

Sensor BLE Board

Bluey is an Open Source BLE (Bluetooth Low Energy) development board with Temperature, Humidity, Ambient Light and Accelerometer sensors.

More Information

iCE Bling

FPGA Wearable

iCE Bling is an Lattice iCE40UP5k FPGA based LED earrings. The earrings have 8x8 grid LEDs and are powered by CR2032 coin cell.

More Information

PogoProg

Programming Adapters

PogoProg is a range of spring-loaded programming adapters from Electronut Labs, designed with aesthetics and ergonomics in mind for repeated programming and test workflows.

More Information

HackaBLE

Tiny BLE Board

hackaBLE is a tiny (~ 18 mm x 28 mm) Open Source Nordic nRF52832 based BLE development board you can embed in your BLE projects.

More Information

Bumpy

SWD Debugger

Bumpy is an inexpensive Open Source blackmagic probe compatible SWD debugger designed to be used with ARM GDB. It supports many platforms, but was primarily designed for use with our Nordic Semiconductor nRF BLE boards.

More Information

Papyr

Low-Power Connected Display

Papyr combines e-paper with low-power wireless technology. Based on the Nordic nRF52840, it supports BLE and mesh protocols like Thread, BLE Mesh, and Zigbee for applications such as dynamic price tags and sensor displays.

More Information

Blip

BLE / 802.15.4 Development Board

Blip is a development board for BLE and 802.15.4 wireless applications, based on the Nordic nRF52840. It includes a built-in Black Magic Probe debugger, sensors, and SD card support for low-power prototyping.

More Information

Indrium Pi HAT

Wireless Gateway

Indrium Pi HAT is a Nordic nRF52840 shield for Raspberry Pi 3b+ that converts it into a low-power wireless gateway for BLE and 802.15.4 networks, with NFC for device commissioning.
